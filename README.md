# NTDLL Suspend/Auto Suspend (Open Source)

**Great For Reverse Engineering**

This Program Will Allow For Suspending, Resuming And Continuous Suspending On Any Program's PID

Example 1:

```
PID: 12154
[S] Suspend, [R] Resume, [CS] Continuous Suspend: s
Found Module (ntdll.dll)
Suspending PID 12154...
```

Example 2:

```
PID: 12154
[S] Suspend, [R] Resume, [CS] Continuous Suspend: r
Found Module (ntdll.dll)
Resuming PID 12154...
```

Example 3:

```
PID: 12154
[S] Suspend, [R] Resume, [CS] Continuous Suspend: cs

(Clears Console)

Delay: 500
Found Module (ntdll.dll)
Suspending PID 12154...

(Loop Begins)
```
